Instructions

Welcome to the Synthetic Data Generator! Please upload a CSV file with the following columns: field_name (name of the field), data_type (e.g., integer, float, string), max_value (maximum value for numeric types), min_value (minimum value for numeric types), and probabilistic_distribution (e.g., normal, uniform). Example: 

            field_name,data_type,max_value,min_value,probabilistic_distribution
            age,integer,100,0,uniform
            salary,float,200000,30000,normal
            name,string,,,,
            start_date,date,,,,

Upload the CSV file to start generating synthetic data. Ensure your CSV has no missing headers or typos. If you need help, check the FAQ or contact support.
