Instructions

Welcome to the Synthetic Data Generator! Please upload a Json file with the following columns: field_name (name of the field), data_type (e.g., integer, float, string), max_value (maximum value for numeric types), min_value (minimum value for numeric types), and probabilistic_distribution (e.g., normal, uniform). Example:

field_name,data_type,max_value ,min_value ,probabilistic_distribution

Upload the Json file to start generating synthetic data. Ensure your Json has no missing headers or typos. You can also attach additional instructions for the llm in the remarks section.
